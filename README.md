# College Event Finder (Simple)

A simple **College Event Finder** website built with **HTML, CSS, and JavaScript**. Students can:

- View upcoming events
- Register for events (saved to **localStorage**)
- Get reminders:
  - **Download `.ics`** calendar files (works with Google Calendar / Outlook / Apple Calendar)
  - Optional **browser notifications** while the page is open

## Run

1. Open `index.html` in a browser.
2. Use **Register** to sign up for an event.
3. Click **Download .ics** to add the event to your calendar.

## Notes

- Data is stored locally in your browser (this is not a multi-user backend).
